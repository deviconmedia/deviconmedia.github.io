# <img src="https://via.placeholder.com/40?text=Icon" alt="Logo Iconmedia" style="vertical-align: middle; margin-right: 10px;"> Dokumentasi Resmi Developer
## PT. Iconmedia Nusantara Abadi

Selamat datang di pusat dokumentasi resmi yang dikelola langsung oleh **Tim Developer PT. Iconmedia Nusantara Abadi**.

## Overview
Situs ini adalah sumber informasi tunggal (Single Source of Truth) untuk seluruh project, API, kebijakan teknis, serta catatan perubahan sistem (changelog) di lingkungan perusahaan.

---

## 🎯 Tujuan Dokumentasi

Dokumentasi ini dibuat untuk memastikan seluruh tim, baik internal maupun mitra pengembang, memiliki akses terhadap informasi teknis yang:

- **Akurat & Terkini:** Selalu diperbaharui seiring dengan rilis sistem terbaru.
- **Terstruktur:** Mulai dari panduan memulai (onboarding) hingga referensi teknis mendalam.
- **Tertelusuri:** Mencatat setiap perubahan sistem melalui halaman Changelog.

---

## 📚 Mulai Menggunakan Dokumentasi

Gunakan menu navigasi di sebelah kiri (atau tombol ☰ pada perangkat seluler) untuk menjelajahi topik-topik berikut:

| Section | Isi |
| :--- | :--- |
| **Panduan Developer** | Cara setup environment, standar coding, dan alur kerja Git. |
| **Dokumentasi API** | Endpoint, autentikasi, skema request/response untuk seluruh API. |
| **Infrastruktur** | Arsitektur server, basis data, dan prosedur deployment. |
| **HRIS Changelog** | Catatan rinci mengenai update, perbaikan bug, dan fitur baru pada sistem HRIS. |
